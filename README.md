# AI-Powered Predictive Maintenance System 🛠️

This project predicts industrial machine failures using IoT sensor data like Temperature, Vibration, and Current. 

## 📊 Project Highlights
- **Real-time Visualization:** Dynamic graphs for sensor telemetry.
- **Smart Predictions:** Instant alerts for 'Normal' vs 'Failure' states.
- **Tech Stack:** Python, Streamlit, Scikit-learn, Pandas, Joblib.

## 🚀 How to use
1. Clone the repo.
2. Install requirements: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
